# 📰 News-Summarizer

AI powered news summarizer app using Python.

## Features
- Fetch latest news using News API
- Generate short AI summary of any article
- Clean and simple interface with Streamlit

## Tech Stack
`Python` `Streamlit` `News API` `Google Generative AI`

## How to Run
1. Clone the repo
   ```bash'
   git clone https://github.com/priyasln001/News-Summarizer.git
   ```
 
 2. Install requirements
 ```bash
 pip install -r requirements.txt
 ```
 
 3. Create .env file and add API keys
 ```bash
 NEWS_API_KEY=your_newsapi_key
 GEMINI_API_KEY=your_gemini_key
 ```

 4. Run the app
 ```bash
streamlit run app.py
 ```
